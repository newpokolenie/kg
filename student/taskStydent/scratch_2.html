Лабиринт

Для выполнения данного задания понадобится:

  • Скачать картинку лабиринта и установить ее как фон.
  • Создать спрайт главного персонажа, который будет ходить по лабиринту.
  • Создать 9 спрайтов ключей.
  • Создать спрайт двери.
  • Создать переменные «Ключи» и «Время».
  1. При нажатии на флажок главный персонаж появляется вначале лабиринта. Сделать для него управление влево, вправо, вниз, вверх используя стрелки. Кот не должен проходить через стены лабиринта.
  2. Создаем спрайт двери (прямоугольный блок), у которого будет два костюма: первый костюм будет черного цвета, второй костюм будет зеленого цвета. Дверью перекрываем выход из лабиринта. Когда нажимаем флажок, дверь меняет свой костюм на черный, а когда все ключи собраны, дверь меняет свой костюм на зеленый.
  3. Спрайты ключей расположить по всему лабиринту. Если ключи касаются кота, то они прячутся и увеличивают переменную «Ключи» на 1. Когда флажок нажимается, ключи должны появиться на своих местах.
  4. Создать фоны «Вы проиграли» и «Вы победили»
  5. В начале игры переменная «Время» = 300 и каждую секунду изменяется на -1.
  6. При нажатии на флажок переменная «Ключи» = 0, а переменная «Время» = 300. Если время <1, то появляется фон «Вы проиграли», если за отведенное время вы собрали все ключи и касаетесь зеленой двери, то появляется фон «Вы победили».
