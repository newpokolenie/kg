Создание выпадающих менюшек

Новый селектор

Дочерний селектор > говорит о том что мы выбираем только тот элемент который является ребенком для основного селектора.

Пример:

  1. body>p {
  2. color: red;
  3. }

В данном примере говорится о том, что текст станет красным только в тех p которые находятся внутри body, а те p которые находятся внутри других элементов, красными не станут.

Как создать выпадающее меню

Мы с вами уже почти вошли во вкус живой верстки - сегодня мы ее распробуем основательно на примере выпадающего меню, которое сверстаем на чистом html+css, без применения скриптов.

Итак, у нас есть спец. задание:

В шапке страницы есть меню, из которого по наведению должно выпадать еще одно маленькое подменю. Давайте попробуем его сверстать.

Начинаем как всегда, с того, что мы уже знаем - с разметки для шапки:

  1. <div class="header">
  2. <div class="container">
    1. <ul class="menu">
    2. <li><a href="#">Home</a></li>
    3. <li><a href="#">Plans and Services</a></li>
    4. <li><a href="#">Infrastructure</a></li>
    5. <li><a href="#">News</a></li>
    6. <li><a href="#">About</a></li>
    7. <li><a href="#">Support</a></li>
    8. </ul>
  4. </div>
  5. </div>

Пока ничего сложного, правда? Давайте добавим немного стилей для всей страницы

  1. /********Global********/
  1. body>p {
  2. font-family: 'OpenSans', Arial, sans-serif;
  3. font-size: 12px;
  4. background: #6c2f84; /*сиреневый фон*/
  5. margin: 0;
  6. padding: 0;
  7. }
  1. .container{
  2. width: 100px;
  3. margin: 0 auto;
  4. }

и для меню:

  1. /********Header********/
  1. .header {
  2. min-height: 97px;
  3. background: #fff;
  4. }
  1. .header .menu {
  2. margin: 0 0 0 156px;
  3. padding: 0;
  4. list-style: none;
  5. }
  1. .header .menu li {
  2. display: inline-block;
  3. padding: 20px 28px 18px;
  4. }
  1. .header .menu li a {
  2. display: inline-block;
  3. color: #434343;
  4. font-style: 18px;
  5. text-decoration: none;
  6. padding: 16px 0 19px;
  7. }

В браузере у нас появилось наше обычное меню:

Итак, вся предварительно-подготовительная разметка готова, и мы начинаем верстку выпадашки. Само подменю будет маркированным списком, вложенным в пункт основного списка. Звучит страшновато, но на самом деле все просто:

  1. <ul class="menu">
  2. <li><a href="#">Home</a></li>
    1. <li>
    2. <a href="#">Plans and Services</a>
      1. <ul class="submenu">
      2. <li><a href="#">Dedicated Servers</a></li>
      3. <li><a href="#">VPS Servers</a></li>
      4. <li><a href="#">Shared Hosting</a></li>
      5. <li><a href="#">Colocation</a></li>
      6. </ul>
    4. </li>
  4. <li><a href="#">Infrastructure</a></li>
  5. <li><a href="#">News</a></li>
  6. <li><a href="#">About</a></li>
  7. <li><a href="#">Support</a></li>
  8. </ul>

Важно положить выпадашку ровно в тот пункт li, под которым он должен появляться. Также полезно задать ему дополнительный класс, как бы намекающий на то, что это второстепенное или выпадающее меню: dropdown-menu, submenu, или как-то аналогично. В браузер в этот момент лучше не смотреть - все сломалось. Но это правильно, так и должно быть, ведь пока мы не задали других стилей для нового меню, применяются все стили от родительского меню.

Пора сделать нашего малыша-выпадашку совсем самостоятельным, со своими стилями:

Сначала мы выводим его из основного потока и позиционируем относительно родителя (li)

  1. .header .menu li {
  2. display: inline-block;
  3. padding: 20px 28px 18px;
  4. position: relative;
  5. }
  1. .header .submenu {
  2. background: red;
  3. position: absolute;
  4. top: 70px;
  5. left: 0;
  6. width: 212px;
  7. }

В браузере уже появляется что-то, похожее на правду

Осталось добавить красивых стилей для выпадашки(отступы у пунктов меню, скругления углов, тень вокруг блока и позиционирование пиксель-в пиксель.)

  1. .header .submenu {
  2. /*Цвет заднего фона*/
  3. background: #fff;
  4. /*Позиционирование на странице (относительно li родителя)*/
    1. position: absolute;
    2. top: 93px;
    3. left: 0;
  6. width: 212px;
  7. margin: 0;
  8. padding: 9px 0 11px;
  9. -webkit-border-radius: 3px;
  10. -moz-border-radius: 3px;
  11. border-radius: 3px;
  12. -webkit-box-shadow: 3px 4px 5px 0 rgba(0, 0, 0, 0.15);
  13. -moz-box-shadow: 3px 4px 5px 0 rgba(0, 0, 0, 0.15);
  14. box-shadow: 3px 4px 5px 0 rgba(0, 0, 0, 0.15);
  15. }
  1. .header .submenu li{
  2. display: block; лишки снова занимают всю ширину родителя
  3. padding: 0;
  4. }
  1. .header .submenu li a {
  2. padding: 8px 24px; немного другие отступы у ссылок
  3. }

Для нас это ведь уже совсем не новое действие, с помощью генераторов и глаза-алмаза получаем красивую менюшку в браузере:

Код для меню получился не больше, чем для какой-нибудь навороченной кнопки, а пользы - вагон. Давайте зададим стили на ховер. Наведение мышки мы будеи отслеживать только у li, a стили навешивать на дочернюю ссылку:

Задаем поведения для ссылок основного меню по наведению

  1. .header .menu li:hover>a{
  2. color: #ff4f3b;
  3. padding-bottom: 18px;
  4. border-bottom: 1px solid #fff0ed;
  5. }

Одновременно с этим изменяется поведение и ссылок внутри нашей выпадашки, для них задаем дополнительные стили по наведению:

  1. .header .submenu li:hover a{
  2. padding: 8px 24px;
  3. border-bottom: none;
  4. }

Сейчас в браузере все ровно с наведением, выглядит прилично, не стыдно людям показать, да и самому приятно посмотреть:

Вы наверное, уже заждались, когда же мы будем делать магию появления выпадашки? Этот час настал. Сначала выпадашку надо спрятать (очень логично).

  1. .header .submenu{
  2. display: none;
  3. }

И теперь время взмахнуть волшебной палочкой мышкой и написать заклинание кусочек кода

  1. .header .menu li:hover>.submenu{
  2. display: block;
  3. }

Та-дааа!!! Вы молодец, вы сдали экзамен в школе волшебства css, можете поиграть с выпадашкой в браузере. Теперь можете подобавлять менюшки в другие пункты основного меню и поиграть с ними всеми.
