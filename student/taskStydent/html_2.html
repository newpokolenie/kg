Знакомство с Sublime и создание файла

Sublime Text – текстовый редактор. Применяется при создании html-файлов.

  1. Запустите Sublime Text.
  2. В начале требуется создать новый файл:
  3. В левом верхнем углу File -> New File

! Все html-файлы должны иметь расширение *.html, иначе созданный файл браузер не отобразит.

  1. Далее следует дать наименование и расширение файлу:
  2. Нажимаем сочетание клавиш CTRL + S –> Вводим имя.html -> Сохранить

Для удобства работы желательно создать свою папку и все файлы сохранять в созданную папку. Файлы желательно называть осмысленно, по порядку (file1, file2 и тд) или по названию темы (tags, form и тд).

Задание. Создать отдельную папку. Создать html-файл и сохранить его в уже созданную папку.
