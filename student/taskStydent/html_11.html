Формы

Форма – инструмент, предназначенный для обмена данными между пользователем и сервером.

  • <form>
  • <label>Ключевое слово</label>
  • <input type = “значение формы” name = “имя формы” placeholder = “подсказывающий текст”>
  • </form>

Тег <form> устанавливает форму на веб-странице.

Перед отправкой данных браузер подготавливает информацию в виде пары «имя=значение», где имя определяется атрибутом name тега <input> или другим тегом допустимым в форме, а значение введено пользователем или установлено в поле формы по умолчанию.

Значение вводится в зависимости от того, какая форма требуется:

  • type = “text” – текстовое поле
  • type = “radio” – поле с переключателями
  • type = “checkbox” – поле с флажками
  • type = “password” - поле пароля
  • type = “submit” – кнопка отправки
  • type = “reset“ - кнопка для сброса данных формы
  • type = “number” – цифровое поле
  • type = “email” – email — поле
  • type = “file“ - поле которое позволяет выбирать файл с компьютера и загрузить его в форму
  • type = “date“ - поле которое позволяет вставлять дату в нашу форму

Например, следует реализовать форму:

  • <form>
  • <label>Укажите имя</label>
  • <input type = “text” name = “name”>
  • <label>Введите пароль</label>
  • <input type = “password” name = “pass”>
  • </form>

Задание. Реализовать форму, используя все вышеуказанные типы.
