Проект 5. Приложение переводчик

Переводчик – специальная программа, которая помогает преобразовать текст (слова, предложения и т.д.) с одного языка на другой.

В настоящее время существует большое множество онлайн переводчиков, которые несомненно облегчают жизнь в работе с разными языками.

Задание

Создать программу, которая позволит переводить слова на разные языки.

Требования:

  • Создать кнопку для ввода слова с помощью голоса
  • Создать кнопку выбора языка
  • Создать кнопку перевода
  • Создать кнопку для голосового воспроизведения перевода

Пользователь вводит слово с помощью голоса, далее выбирает желаемый язык. При нажатии на кнопку перевода получает желаемый результат.

Компоненты:

ЯндексПереводчик, ТекстВРечь, Кнопки, Текст, РаспозновательРечи
