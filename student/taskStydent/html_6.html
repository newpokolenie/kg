Работа с картиками в HTML

Давайте теперь разберемся с тем, как разместить изображение на странице вашего сайта. Для этого предназначен тег , имеющий обязательный атрибут src, в котором следует задавать путь к файлу картинки. При этом сам тег не требует закрывающего тега.

В тегах также могут размещаться атрибуты - специальные команды, которые расширяют действие тега.

Атрибуты размещаются внутри открывающего тега в таком формате:

<тег атрибут1="значение" атрибут2="значение">

Кавычки могут быть любыми - одинарными или двойными, допустимо их вообще их не ставить, если значение атрибута состоит из одного слова (но это не желательно).

Атрибуты которые доступны для тега <img>

  • src - атрибут src задает путь к изображению

    Синтаксис src=”путь к картинке”

  • alt - атрибут показывает альтернативный текст, если картинка по каким-то причинам не прогрузилась

    Синтаксис alt=”Текст который покажется вместо картинки ”

  • width - атрибут позволяет изменять ширину картинки

    Синтаксис width = “1+”

  • height - атрибут позволяет изменять высоту картинки
