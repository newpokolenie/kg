Работа со свойством transition

Переходы CSS позволяют изменять значения свойств плавно (от одного значения к другому) за заданную длительность.

Свойства которые применяются для создания переходов в CSS:

transition-delay: s ; - указывает задержку в секундах для эффекта перехода
transition-duration: s; - указывает сколько секунд должен длятся переходит
transition-property: название свойства; - указывает имя свойства CSS, для которого используется эффект перехода. Значение all стоит по умолчанию.
transition-timing-function: ease; - задает эффект перехода с медленным запуском, затем быстро, затем закончится медленно

linear - задает эффект перехода с одинаковой скоростью от начала до конца
ease-in - задает эффект перехода с медленным запуском
ease-out - задает эффект перехода с медленным концом
ease-in-out - задает эффект перехода с медленным запуском и концом

transition: название свойства | длительность | эффект перехода | задержка; - универсальное свойство которое включает в себя сразу все свойства которые применяются для создания эффекта перехода

Эффект перехода срабатывает между двумя состояниями элемента, они могут быть определены с помощью псевдокласса :hover и :active.

Пример
В следующем примере показан элемент 100px * 100px Red

. Элемент
также указал эффект перехода для свойства Width с длительностью 2 секунды:

  • div{
  • width:100px;
  • height:100px;
  • background:red;
  • -webkit-transition: width 2s; /*Safati*/
  • transition: width 2s;
  • }

Теперь давайте укажем новое значение свойства Width, когда пользователь наносит указатель мыши на элемент div:

  • div:hover{
  • width:300px;
  • }

Попробуйте вставьте этот код к себе в редактор и посмотрите на эффект. Красный квадрат в течении 2 секунд будет увеличиваться с ширины 100px до 300px

Пример
В данном примере мы заданим эффект перехода одновременно для свойства Width и Height с длительностью в 2 секунды для ширины и 4 секунды для высоты

  • div{
  • width:100px;
  • height:100px;
  • background:red;
  • -webkit-transition: width 2s, height4s;
  • transition: width 2s, height4s;
  • }

При наведении ширина и высота будут стремиться к 300px

  • div:hover{
  • width:300px;
  • height:300px;
  • }

Опробуйте данный код в редакторе.

Практика

Задание 1.
Создайте синий квадрат шириной и высотой 100px. При наведении квадрат должен плавно увеличить ширину и высоту до 300px, а также сменить цвет на желтый. Задержку и время действия можете выбрать сами.

Задание 2
Создайте круг который будет плавно увеличиваться. Размер можете выбрать по своему усмотрению.

Задание 3
Вернитесь к заданию где вы делали выпадающие менюшки и сделайте так чтобы выпадение происходило плавно.
