Знакомство с CSS и первые свойства

CSS – Cascading Style Sheets (Каскадные Таблицы Стилей) - называются так, потому что HTML-элементы которые распологаются внутри другого HTML-элемента могут наследовать CSS свойтсва своих родителей.

CSS – нужен для того чтобы задавать свойства для HTML–элементов. Свойствами html элементов являются цвет текста, цвет заднего фона, размер текста, вид текста и многие другие.

Существует три способа подключения CSS к HTML-элементам:

Первый способ заключается в том, что css свойства записываются в отдельный файл и подключается через специальный тег .

Преимущества такого подхода в том, что CSS файл может быть один, а HTML документов может быть любое количество. Сделав изменение в одном месте CSS файла, эти изменения применсять ко всем HTML файлам к которым подключен CSS.

Чтобы подключить CSS файл к HTML странице, в теге

<head>

нужно прописать такую конструкцию:

<link rel="stylesheet" href="путь к CSS-файлу">

. CSS файл должен иметь расширение ".css". Обычно его называют "styles.css" или "style.css".

  1. <!DOCTYPE html>
  2. <html>
    1. <head>
    2. <meta charset="UTF-8">
    3. <title>Это заголовок страницы</title>
    4. <link rel="stylesheet" href="style.css">
    5. </head>
    1. <body>
    2. <p>
    3. Это абзац с текстом.
    4. </p>
    5. </body>
  3. </html>

Можно подключать несколько css файлов к одному HTML-документу.

  1. <!DOCTYPE html>
  2. <html>
    1. <head>
    2. <meta charset="UTF-8">
    3. <title>Это заголовок страницы</title>
    4. <link rel="stylesheet" href="style1.css">
    5. <link rel="stylesheet" href="style2.css">
    6. <link rel="stylesheet" href="style3.css">
    7. </head>
    1. <body>
    2. <p>
    3. Это абзац с текстом.
    4. </p>
    5. </body>
  3. </html>

Второй способ заключается в том, что css можно разместить стили внутри тега

<style>

, который пишется внутри тега

<head>

. Недостаток этого способа в том что стили можно применять только для одного HTML–документа. Этот способ не рекомендуется.

  1. <!DOCTYPE html>
  2. <html>
    1. <head>
    2. <meta charset="UTF-8">
    3. <title>Это заголовок страницы</title>
    4. <style>
      1. p{
      2. color: red;
      3. }
    6. </style>
    7. </head>
    1. <body>
    2. <p>
    3. Это абзац с текстом.
    4. </p>
    5. </body>
  3. </html>

Третий способ заключается в том, что css можно добавить для каждого тега при помощи атрибута "style".

  1. <!DOCTYPE html>
  2. <html>
    1. <head>
    2. <meta charset="UTF-8">
    3. <title>Это заголовок страницы</title>
    4. </head>
    1. <body>
    2. <p style="color: color: red; font-weight: bold;">
    3. Это абзац с текстом.
    4. </p>
    5. </body>
  3. </html>

Существует такое правило: если что-то можно сделать первым способом, то использовать второй и третий способ дурной тон.

Общее CSS правило - это так мы должны писать текст в CSS документе

  1. селектор {
  2. свойство: значение свойства;
  3. }

Каждому тегу в HTML соответствует так называемый селектор CSS. Селектор это элемент которому мы задаем css свойство. От англ.слова select – вбирать.

Существует три вида основных селекторов: по классу, по идентификатору(id), по названию тега.

По названию тега – это такой селектор когда в css мы должны писать название тега для которого хотим задать свойства, если в HTML мы ставили стрелочки <>, то в css их ставить не нужно.

Пример : зададим для всех наших параграфов цвет текста красный:

  1. .p {
  2. color: red;
  3. }

По названию класса нашего тега – в html мы можем задать класс для наших тегов с помощью атрибута class="class_name", одно название класса мы можем применять для нескольких HTML элементов. Когда мы используем селектор по классу, то в CSS перед названием класса нужно поставить точку: .class_name.

Пример : у нашего тега p будет класс paragraph зададим для всех параграфов у которых есть класс paragraph задать цвет текста красный:

  1. <--HTML-->
  2. <p id="peshka"></p>
  1. /* CSS */
  2. #peshka {
  3. color: red;
  4. }

Первые свойства css и их значения

Свойство которое позволяет нам задать цвет нашему тексту – color. Значения для этого свойства можно задать тремя способами: английское название тега, rgb(0-255,0-255,0-255), #000000.

Пример:

  1. .first {
  2. color: red;
  3. }
  1. .p {
  2. color: rgb(255, 236, 114);
  3. }
  1. .p {
  2. color: #ff0000;
  3. }

Для задания ширины и высоты элементов в CSS предназначены свойства width и height. Значения этих свойств, как правило, задаются в пикселях. Пиксели обозначаются как px и добавляются к числу.

Есть таблица и для ее ячеек мы хотим задать ширину 200px и высоту 100px

  1. <table border="1">
    1. <tr>
    2. <td>1</td>
    3. <td>2</td>
    4. <td>3</td>
    5. </tr>
    1. <tr>
    2. <td>4</td>
    3. <td>5</td>
    4. <td>6</td>
    5. </tr>
    1. <tr>
    2. <td>7</td>
    3. <td>8</td>
    4. <td>9</td>
    5. </tr>
  5. </table>

В CSS фале прописываем следующий код:

  1. td {
  2. width: 200px;
  3. height: 100px;
  4. }

Так будет выглядеть таблица в браузере

Сейчас научимся выравнивать текст. Для этого используется свойство text-align. Текст можно выравнять по левому краю (значение left), по правому (значение right), по центру (значение center) и одновременно и по правому, и по левому краю (значение justify).

  1. p {
  2. text-align: left; //обычно стоит везде по умолчанию
  3. }
  1. p {
  2. text-align: right;
  3. }
  1. p {
  2. text-align: center;
  3. }
  1. p {
  2. text-align: justify; //работу этого свойства можно увидеть, если текст в параграфе будет достаточно большим
  3. }

Например:

  1. <p> очень длинный текст…. </p>
  1. p {
  2. width: 300px;
  3. text-align: justify;
  4. }

Как видите текст выравнен одновременно и по левому и по правому краю
