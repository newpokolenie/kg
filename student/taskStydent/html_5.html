Работа со списками в HTML

Список – совокупность упорядоченных записей, написанных в столбик.

Списки делятся на три группы: нумерованные, маркированные, вложенные.

Нумерованный:

  1. Компьютер
  2. Клавиатура
  3. Процессор

Маркированный:

  • Компьютер
  • Клавиатура
  • Процессор

Вложенный:

  • Компьютер
    • Клавиатура
    • Мышка

Структура нумерованного списка:

  1. <ol> <--тег который создает нумерованный список-->
  2. <li> … </li><--тег который является элементом списка-->
  3. ……
  4. </ol>

Структура маркированного списка:

  1. <ul><--тег который создает маркированный список-->
  2. <li> … </li>
  3. ……
  4. </ul>

Вложенный список – список в списке.

  1. <ol>
  2. <li>Компьютер
    1. <ol>
    2. <li>Клавиатура</li>
    3. <li>Мышка</li>
    4. </ol>
  3. </li>
  4. </ol>
