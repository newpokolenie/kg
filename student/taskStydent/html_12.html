Нестандартные формы

Select – выпадающее меню.

Если количество вариантов для выбора занимает слишком много места, вы можете воспользоваться выпадающими меню <select>.

  • <select>
  • <option>Значение1</option>
  • <option>Значение2</option>
  • </select>

Textarea – многострочный текст. Может быть изменён размер поля.

  • <textarea>
  • …
  • </textarea>

Fieldset - предназначен для группирования элементов формы. Такая группировка облегчает работу с формами, содержащими большое число данных. Например, один блок может быть предназначен для ввода текстовой информации, а другой — для флажков.

  • <form>
    • <fieldset>
    • <legend> Ключевое слово</legend>
    • <input ……>
    • </fieldset>
  • </form>

<button> - создает кнопку на вашей веб-странице у него есть также атрибут type. Кнопка также отправляет данные на сервер из формы. По своей работе схож с input у которого тип submit и reset. Отличие в том, что кнопку лучше поддается стилизации с помощью css.

  • <form>
  • <button type=”submit”>текст кнопки</button>
  • <button type= “reset”>текст кнопки</button>
  • </form>
