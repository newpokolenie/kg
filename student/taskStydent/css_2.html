Свойства о работе с текстом в CSS

Жирность текста:

  1. font-weight: bold; //жирный текст
  2. font-weight: normal; //обычный текст

Курсивный текст:

  1. font-style: italic; //курсивный текст
  2. font-style: normal; //обычный текст

Размер шрифта:

  1. font-size: 30px;

Расстояние между строк:

  1. line-height: 10px;

Красная строка:

  1. text-indent: 30px;

Подчеркивание текста:

  1. text-decoration: underline; //подчеркивает текст
  2. text-decoration: overline; //подчеркивает текст сверху
  3. text-decoration: line-through; //перечеркивает текст
  4. text-decoration: none; //убирает подчеркивание

Вертикальное выравнивание:

  1. vertical-align: top; //выравнивание элемента по верху
  2. vertical-align: bottom; //выравнивает элемент по низу
  3. vertical-align: middle; //выравнивает элемент по середине
  4. vertical-align: sub; //элемент изображается как подстрочный
  5. vertical-align: super; //элемент изображается как надстрочный

Изменения шрифта текста в CSS – свойство font-family позволяет нам менять вид нашего текста на странице. Существует два вида текста с засечками (Тут есть засечки) и без засечек (Тут нет засечек). Засечки это маленькие выступы на концах букв.

Шрифты значения которые будут представлены ниже называются web-безопасными шрифтами, так как есть почти у каждого человека на компьютере. Ведь если шрифта на компьютере не будет то установится шрифт по умолчанию. Обычно это Times New Roman.

Существует большое многообразие различных шрифтов.

Практика

Перед выполнение задания на свойства создайте следующий текст на html странице:

Задание 1

  • Сделайте все <td> жирным.
  • Сделайте <h1> нежирным.
  • Сделайте одновременно <th>, <h1> и <h2> нежирным.

Задание 2

  • Сделайте все <h2> курсивом.
  • Сделайте все абзацы <p> курсивом, а первый абзац - нет.

Задание 3

  • Задайте для всех <h2> размер текста 20px.
  • Задайте для всех абзацев <p> размер текста 15px.

Задание 4

  • Сделайте для абзацев <p> шрифт Arial.
  • Сделайте для <h2> шрифт Times New Roman.
  • Сделайте для <h3> любой шрифт без засечек.

Задание 5

  • Задайте расстояние между строк для абзацев <p> в 30px.

Задание 6

  • Задайте красную строку для всех абзацев 30px, для второго абзаца <p> уберите красную строку.

Задание 7

  • Поставьте текст в таблице <table> по верхнему краю по вертикали.
  • Поставьте текст в <th> по центру по вертикали.
