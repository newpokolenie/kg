Знакомство с HTML

HTML (Hyper Text Markup Language) – язык гипертекстовой разметки.

! HTML не является языком программирования.

Данный язык предназначен для создания структуры html – файлов.

Данный язык используется для отображения элемент–объектов на веб-странице. Таким образом он отвечает за структуру и содержание веб-страниц(html-страниц) из которых состоят сайты Интернета.

HTML является теговым языком, так как состоит из тегов.

На данный момент выпущено пять версий языка. Первая версия была разработана между 1986 и 1991 годами, а последняя (5.2) — в 2017. Изначально он должен был стать независимым от каких-либо платформ — отображаться везде одинаково. Но этого не случилось, потому что у пользователей росли требования к мультимедиа.
