Работа с ссылками и задним фоном

Состояние ссылок

Текст после двоеточия(:) у селектора называется псевдоклассом, например :hover

Псевдоклассы позволяют отлавливать разные состояния элементов.

  1. a:link //состояние ссылки до того как ее посетили
  2. a:hover //состояние ссылки когда на нее навели курсором
  3. a:active //состояние ссылки когда на нее нажали мышкой, но еще не отпустили
  4. a :visited //состояние ссылки когда по ней уже перешли

Пример:

  1. <a href="#">Ссылка</a>
  1. a:link {
  2. color: red; //в обычном состоянии ссылка будет красной
  3. }
  1. a:visited {
  2. color: green //после перехода по ссылке она станет зеленой;
  3. }
  1. a:hover {
  2. text-decoration: none; //при наведении у ссылки будет исчезать подчеркивание
  3. }
  1. a:active {
  2. color: blue; //при нажатии на ссылку она станет синей
  3. }

Псевдоклассы наследуют состояния друг друга. Например: Если убрать подчеркивание для состояния :link, то подчеркивание уберется для всех состояний.

Из-за наследования следует размещать псевдоклассы в документе css в таком порядке: :link, :visited, :hover, :active. Если какой-то псевдокласс не нужен его можно не писать.

Часто состояния :link и :visited объединяют через запятую:

  1. a:link, a:visited{
  2. color: red;
  3. }
  1. a:hover{
  2. text-decoration: none;
  3. }
  1. a:active{
  2. color: blue;
  3. }

Работа с фоном

  1. background-color: цвет; //выбирается любым из трех способов
  1. background-image: url("путь к картинке"); //задает фоновую картинку

Фоновая картинка заполняет собой весь блок, это можно исправить следующим свойством:

  1. background-repeat: no-repeat //картинка не будет повторятся и займет левый верхний угол
  2. background-repeat: repeat-x //картинка будет повторяться по координате x (по горизонтали)
  3. background-repeat: repeat-y //картинка будет повторяться по координате y (по вертикали)

Для того чтобы менять положение нашей фоновой картинке на экране или в блоке используется свойство :

  1. background-position: позиция x позиция y
  2. позиция x : left, right, center, %, px
  3. позиция y: top, center, bottom, %, px
  4. background-size: %, px;

К примеру свойство: background-position: right bottom поставит фоновую картинку в правый нижний угол

Вместо ключевых слов мы может также использовать px и %

Например: background-position: 90% 30px; наш фон на 90% от ширины блока сдвинутся вправо и на 30px вниз.

Можно также объединять значения в пикселях и ключевые слова: background-position: 90% center;

Для того чтобы разрешить или запретить прокручивать фон в месте с текстом на странице существует свойство background-attachment

  1. background-attachment: scroll; //позволяет прокручивать фон вместе с текстом
  2. background-attachment: fixed; //делает фон фиксированным (запрещает прокручивать фон вместе с текстом)

Задание 1

Создать ссылку которая при состоянии :link и :visited будет синяя, а при состоянии :hover(наведении) должна будет стать зеленой

Задание 3

Задание 10
