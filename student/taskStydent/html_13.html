Семантика и HTML5 теги

Семантика – это изучение значений слов и фраз на языке.

Семантические элементы = элементы с смыслом.

Многие веб-узлы содержат HTML-код, например: <div ID = "NAV"< <div class = "заголовок"< <div ID = "нижний колонтитул">

для обозначения навигации, верхнего и нижнего колонтитулов.

HTML5 предлагает новые семантические элементы для определения различных частей веб-страницы:

<article></article> Тег служащий для обозначения статьи на сайте
<aside></aside> Тег который нужен для того чтобы отобразить боковые колонки на сайте
<footer></footer> Оперделяет нижний колонтикул страницы, также его называют “подвалом” страницы
<header></header> Задает заголовок для документа или раздела, акже называют шапкой странице, в ней обычно распалагается логотип и навигация
<main></main> Указывает основное содержимое страницы
<nav></nav> Определяет навигационные ссылки на странице
<section></section> Определяет раздел на странице, обычно вставляется внутри тега main
<time></time> Определяет дату и время
