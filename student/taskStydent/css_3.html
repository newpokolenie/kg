Работа с границами

Контейнеры

Познакомимся с двумя новыми тегами:

<div> </div> - блочный контейнер

<span> </span> - строчный контейнер

Эти два тега являются контейнерами, т.е при добавлении их в html страницу, никакого визуального эффекта они не дадут, но зато в них можно вставлять другие теги или текст. Также эти два тега очень удобно стилизовать.

Тег <div> является блочным контейнером и внутрь него можно добавлять любые другие теги.

Тег <span> является строчным контейнером и в него можно вставить только текст, который нужно будет стилизовать особенным образом.

Далее для выполнения практических задач, вам нужно будет применять тег <div>

Работа с границами CSS

  1. border-width: размер; //свойство позволяющее задать толщину нашей границы
  2. border-color: цвет; //свойство задающее цвет нашей границы
  3. border-style: solid; //сплошная граница
  4. border-style: dotted; //граница в виде точек
  5. border-style: dashed; //граница в виде пунктиров
  6. border-style: double; //двойная граница
  7. border-style: none; //отсутствие границ

Сокращенное свойство для границ border: толщина стиль цвет

  1. border: 1px solid black;

Границы для отдельных сторон

  1. border-left //граница будет только слева
  2. border-right //граница будет только справа
  3. border-top //граница будет сверху
  4. border-bottom //граница будет снизу
  1. border-top: 1px dotted blue;

Скругление углов границы

  1. border-radius: 10px;

lt – левый верхний угол

rt – правый верхний угол

rb – правый нижний угол

lb - левый нижний угол

  1. border-radius: 10px 20px 30px 40px; //ltpx rtpx rbpx rlpx
  2. border-radius: 20px 30px; //ltrbpx rtlbpx
  3. border-radius: 30px 20px 10px; //ltpx rtlbpx rbpx

Для того чтобы сделать круг

  1. border-radius: 100% //с учетом того что наш блок является квадратом
