Работа со списками и таблицами

Работа с маркерами списка

Свойство: list-style-type

Значения для списка ul:

  1. square //квадрат
  2. circle // незакрашенные круги
  3. disc //закрашенные круги – стоят по умолчанию
  4. none //убрать маркеры

Значения для списка ol:

  1. lower-roman //маленькие римские цифры
  2. upper-roman //большие римские цифры
  3. lower-alpha //маленькие латинские буквы
  4. upper-alpha //большие латинские буквы
  5. decimal //арабские числа 1, 2, 3 и т.д
  6. decimal-leading-zero //арабские числа начиная с 0
  7. none //убирает маркеры
  8. lower-greek //греческие символы

Сделать маркер в виде картинок:

  1. list-style-image: url("ссылка на картинку");

Позиция маркеров в списке:

По умолчанию маркеры списка не входит в сам список. Для того чтобы изменить положение маркеров используется свойство list-style-position

  1. inside //внутри списка
  2. outside //снаружи списка
